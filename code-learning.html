Code Learning Websites

Programming skills are in high demand, whether you work for a for-profit or a not-for-profit organization. Learning how to program is easier than ever with a variety of interactive platforms. Here is a partial list of some of them:

Codecademy


https://codecademy.com/

With courses in 9 different languages, which are among the most widely used front-end languages, Codecademy is a popular site where you can learn how to code for free.

Code School


https://codeschool.com/

With courses in over 50 different languages and tools, Code School offers free introductory access to 10 courses and monthly and yearly rates for premium access to all courses.

Treehouse


https://treehouse.com/

Aimed at teaching programming and business skills to anyone from beginners to experts, Treehouse offers basic and pro plans at monthly and annual rates.

Coursera


https://www.coursera.org/

A heavyweight in the world of massively open online learning that partners with top universities, Coursera users can learn top tech skills for free and have the option on some courses to, for small fees, receive verified certificates of completion.
