Bootstrap
by Twitter

Bootstrap is a collection of free to use templates that are bundled together to help developers and product managers create quick responsive clean websites. Originally, bootstrap websites were created to eliminate the need for developers to re-code websites from scratch every time they wanted to create a new one. Today, these templates have been expanded, and you can download very modern looking bootstrap themes to use as a production level website.
